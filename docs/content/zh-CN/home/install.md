绎 基于 GO 开发，你可以使用多种方式安装。

# 从源代码安装

```bash
git clone https://gitcode.com/OpenCangjieCommunity/Yi.git
cd ./Yi
go install
```
绎将被安装至 `~/go/bin` 。
